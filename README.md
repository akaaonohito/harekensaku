# 晴れる屋 MTG 検索アプリ

検索窓に入力したワードを使って、晴れる屋 MTG の検索結果を取得し、
プルダウン形式で表示するシンプルなWebアプリです。

## ファイル構成

```
harekensaku/
├── package.json          # Node.js プロジェクト設定ファイル
├── server.js             # Express サーバー + スクレイピング処理
├── public/
│   ├── index.html        # HTMLメイン画面
│   ├── style.css         # スタイルシート
│   └── script.js         # フロントエンド処理
└── README.md             # このファイル
```

## セットアップ手順

### 前提条件
- Node.js 14 以上がインストールされていること
- npm がインストールされていること

### ステップ

1. **プロジェクトディレクトリに移動**
   ```
   cd d:/物置/Code/harekensaku
   ```

2. **依存パッケージをインストール**
   ```
   npm install
   ```
   以下のパッケージがインストールされます：
   - `express`: Webサーバーフレームワーク
   - `axios`: HTTP通信ライブラリ
   - `cheerio`: HTML解析ライブラリ

## 実行手順

1. **サーバーを起動**
   ```
   npm start
   ```

2. **ブラウザでアクセス**
   ブラウザで以下のURLを開きます：
   ```
   http://localhost:3000
   ```

3. **サーバーの停止**
   ターミナル上で Ctrl+C を押す

## 使い方

1. **検索ワードを入力**
   - 例：「神聖なる泉」「Lightning Bolt」など
   - 晴れる屋で検索可能なワードを入力してください

2. **「検索」ボタンをクリック（またはEnterキーを押す）**
   - 読み込み中メッセージが表示されます

3. **プルダウンでカードを選択**
   - 検索結果のカード名一覧がプルダウンに表示されます
   - プルダウンから1件選択します

4. **「決定」ボタンをクリック**
   - 選択したカードの名前と価格が下部に表示されます

5. **再検索する場合**
   - 新しい検索ワードを入力して、もう一度「検索」をクリック
   - 前回の結果は自動的にクリアされます

## 各処理の説明

### 1. 検索取得処理 (server.js の searchHareruyaMTG 関数)

- 検索ワードをURL形式にエンコード
- 晴れる屋のサイトに GET リクエストを送信
- HTML を cheerio で解析
- 商品要素から、カード名・URL・価格の情報を抽出
- 最大20件までの候補を配列で返す
- 解析に失敗した場合は詳細なエラーメッセージを返す

**特徴：**
- 複数のセレクタパターンで商品を検索（変更に強い構造）
- 複数の価格形式に対応（¥記号パターン、円記号パターン等）
- 相対URLを絶対URLに自動変換

### 2. プルダウン反映処理 (script.js の populateCardSelect 関数)

- サーバーから受け取った結果を配列に格納
- 各カード情報をプルダウンのオプションに追加
- value属性に配列インデックスを設定
- 表示テキストにカード名を使用

### 3. 決定ボタン押下処理 (script.js の handleDecision 関数)

- プルダウンの value（配列インデックス）を取得
- グローバル変数 currentSearchResults から該当するカード情報を取得
- バリデーション：選択状態をチェック

### 4. カード名・価格表示処理 (script.js の displayCardResult 関数)

- カード名を画面下部の専用エリアに表示
- 価格情報があればフォーマット（¥を付与）、なければ「価格情報が取得できませんでした」と表示
- HTML の innerHTML を直接更新して表示

## エラー時の挙動説明

| エラー内容 | 表示メッセージ | 対応策 |
|----------|--------------|------|
| 検索ワード未入力 | 「検索ワードを入力してください」 | 検索ワードを入力してから検索 |
| 検索結果0件 | 「検索結果が見つかりませんでした」 | 別のワードで再検索 |
| 通信エラー | 「通信エラー: （詳細）」 | ネットワーク接続を確認、サーバーが起動しているか確認 |
| スクレイピング失敗 | 「検索結果の取得に失敗しました」 | 晴れる屋のページ構造が変わった可能性あり。サーバー.jsのセレクタを確認・更新 |
| プルダウン未選択で決定 | 「カードを選択してください」 | プルダウンでカードを1件選択してから「決定」をクリック |
| 価格情報が取得できない | 「価格情報が取得できませんでした」 | 晴れる屋の仕様により、価格が取得できないカードもあります |

## トラブルシューティング

### サーバーが起動しない

- Node.js がインストールされてるか確認：`node --version`
- 依存パッケージがインストールされているか確認：`npm list`
- ポート3000 が他のプロセスで使用されていないか確認

### 検索しても結果が表示されない

- ブラウザのコンソール（F12 → コンソール）でエラーメッセージを確認
- サーバーログ（ターミナル）にエラーメッセージがないか確認
- 晴れる屋のサイト構造が変わった可能性あり。セレクタを更新してください

### 価格が表示されない

- 晴れる屋のページ構造変更に対応が不十分な場合があります
- server.js の `pricePatterns` 配列にパターンを追加してください
